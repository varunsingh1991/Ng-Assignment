Assignment

Create an application to display a list of users. This application should have below functionalites.

Each entry should contain the following:

  • User Id (id)
  • Name (firstName + lastName)
  • Age (age)
  • Gender (gender)
  • Email (email)
  • Phone Number (phone)
  • Profile Picture (Image)
  • Company Name (company->name)
  • Address (company->address)
  • Delete button to delete a record.
  • Edit button to edit a user.
  1. There should be 10 entries Per page (limit).
  2. There should be a pagination functionality with ‘Next’ and ‘Previous’ links to navigate between pages. Allow user to toggle between 5, 10 or 25 entries. (On pagination and toggling functionality data should be retrieved from back end)
  3. Filter functionality should also be there to filter users based the given text by the user in search box.
  4. On clicking delete button, a confirmation dialog should open. After user confirmation record should be deleted.
  5. On clicking edit button, a dialog will be opened to show user details in edit mode. User details can be edited from this dialog.
